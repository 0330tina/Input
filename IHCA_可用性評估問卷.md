# IHCA 風險預測平台可用性評估問卷

**適用對象**：臨床醫護人員  
**評估目的**：了解系統操作性、介面清晰度與整體可用性，供醫療資訊系統可用性研究與論文使用。  
**填答說明**：請依您實際使用本平台之經驗勾選，所有題目均為單選。

---

## 第一部分：說明文字（可貼於 Google Form 開頭）

```
【問卷說明】

本問卷旨在評估「IHCA 院內心因性猝死風險預測平台」之可用性，對象為曾使用本系統之臨床醫護人員。請依據您實際操作經驗，依同意程度勾選最符合的選項。填答資料僅供學術與系統改善使用，採匿名方式，不會用於個人識別。

請依以下尺度作答：
1 = 非常不同意
2 = 不同意
3 = 普通
4 = 同意
5 = 非常同意
```

## 學術使用說明

- 本問卷設計參考 System Usability Scale（SUS）之結構與量表形式，並依醫療資訊系統情境調整題數與用語。
- 量表題採 5 點李克特量表，可計算各題平均、標準差，並依面向或總分進行分析。
- 開放式題目可進行質性編碼，作為介面與流程改善之依據。
- 若需與 SUS 分數對照，可於同研究另施測完整 SUS，本問卷作為補充或情境化評估工具。
